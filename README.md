egressão Linear do Zero em Python

Este projeto demonstra como construir uma **rede neural de uma única camada (regressão linear)** usando apenas **NumPy** e **Matplotlib** — sem frameworks de aprendizado de máquina.

O modelo aprende a converter valores em °C para °F a partir de dados gerados com ruído, implementando manualmente o **feedforward**, **cálculo da perda (MSE)** e **backpropagation**.

---

##  Tecnologias utilizadas
- Python 3.10+
- NumPy
- Matplotlib

---

##  Estrutura do código
O script `rede_neural_linear.py` contém:

- Geração de dados sintéticos (`Curva_Linear`)
- Implementação manual do **feedforward**
- Função de **erro quadrático médio (MSE)**
- **Backpropagation** com atualização de pesos e bias
- **Treinamento** com número configurável de épocas
- Visualização dos resultados em um gráfico
